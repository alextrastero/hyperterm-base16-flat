# `hyperterm-base16-flat`

## Chris Kempson's base 16#flat theme for hyperterm.

![hyperterm-base16-flat](https://cldup.com/lmnYCBQP6a.png)

## Install

Add `hyperterm-base16-flat` to `plugins` in `~/.hyperterm.js`.
